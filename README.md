# Network Packet Sniffer with GUI (Python)

## Setup (Windows)

1. Install **Npcap** (recommended: enable WinPcap-compatible mode).
2. Run your terminal as **Administrator** (often required for capture).
3. Install dependencies:

```bash
python -m pip install -r requirements.txt
```

## Run

```bash
python main.py
```

## Default Login

- Username: `admin`
- Password: `admin123`

## Output Files

- `captured_data.log` is created automatically in the project folder.
